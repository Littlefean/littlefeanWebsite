文件操作

  • Ctrl + Shirt + S = 另存为

  • Ctrl + N = 新建

  • Ctrl + O = 打开

高级操作

  • Ctrl + Shift + Alt + T = 自由变换中 等比例复制
  • Ctrl + Shift + ； = 开启自动吸附功能

界面操作

  • Tab = 隐藏菜单栏

  • Shift + Tab = 隐藏部分菜单栏

  • F = 隐藏全部

  • Alt+ 鼠标滚轮 = 放大

  • Ctrl + Tab = 切换图片文件

  • Ctrl + R = 标尺（右键点击标尺可以切换单位）

  • Ctrl + ；= 取消显示标尺

  • 窗口→排列→双排列 = 多界面排列

工具

  • 修补工具：J

选区操作

  • ​ 全选：Ctrl + A

  • ​ 选区工具（L、M、W）、移动选区要用同样的工具，

  • ​ 取消选区：Ctrl + D

  • ​ 选区反选：Ctrl + Shirt + i

  • ​ 默认黑白前景背景色：D

  • ​ 加选：Shift

  • ​ 减选：Alt

  • ​ 交选：Shift + Alt

  • ​ 填充前景色：Ctrl + Delete

  • ​ 填充背景色：Alt + Delete

  • ​ 内容试别：Shift + F5

  • 选区工具：Shift + M

  • 魔棒&选择笔：Shift + W

  • 套索系列工具：Shift + L

  • ​ 得到该图层选区：Ctrl + 点击图层预览图

  • ​ 将路径转化为选区：Ctrl + Enter

  • ​ 描边选区：Alt → E → S

蒙版操作

  • ​ 快速蒙版：Q

  • ​ 剪贴蒙版：Ctrl + Alt + G

图层操作

  • ​ 创建组：Ctrl + G

  • ​ 复制图层：Ctrl + J

  • ​ 合并图层 ：Ctrl + E

  • ​ 合并所有可见图层：Ctrl + Shift + E

  • ​ 盖印所有图层：Ctrl + Shift + Alt + E

路径操作

钢笔工具P

  1. Ctrl

  2. 移动描点

  3. 锚点平滑时，调节调节点，锚点两端平滑

  4. Alt

  5. 得到调节点，锚点两端对称

  6. 调节调节点，锚点两端尖突

路径转化为选区：Ctrl + Enter

路径描边：点路径面板 → 点击下方第二个“用画笔描边路径”

文本操作

  • ​ 文字确认：Ctrl + Enter

  • ​ 细的文字：浑厚

  • ​ 粗的文字：犀利 & 锐利

  • ​ 修改某一行行距：框选本行与上一行进行调整

  • ​ 修改某一个字距：框选本字与前一字进行调整

工具快捷键

  • ​ 画布： Alt + Ctrl + c

  • ​ 图像：Alt + Ctrl + i

界面颜色调整

  • ​ 色阶：Ctrl + L

  • ​ 曲线：Ctrl + M

  • ​ 色相饱和：Ctrl + U

  • ​ 色彩平衡：Ctrl + B //最后一步细节上的微调

  • ​ 黑白：Ctrl + Shift + Alt + B //最专业的调整黑白照片的方式

注意事项

  • ​ 填充颜色之前一定要注意新建图层
