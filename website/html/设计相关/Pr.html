字幕

视频快速加字幕

  1. 转字母网站：https://jianwai.netease.com/

  2. 进入网站→新建项目→语音转写→添加音频

  3. 做好后点击网页内下方的播放按钮，同时校对

  4. 导出str字母文件，导入到pr

  5. 调整整体文字大小：选中所有字幕，更改字体大小

快速加字幕2

  1. 准备好视频、字幕的txt

  2. 打开ArcTime

  3. 字幕的txt格式设置成一行一行的

  4. 复制文字到ArcTime右边里面的地方

  5. 点击 “鼠标+矩形”样子的一个工具，然后左键选取时间轴上的一段，此地方为此句的音频、

  6. 选择完后全部选中字幕，然后点右边的“A”按钮，修改字幕的样式

  7. 导出 到Premiere Pro XML+PNG序列

  8. 弹出窗口中左侧画面高度什么的与工程序列保持一致，修改应用软件为Premiere

  9. 弹出窗口中右边修改字体、描边、阴影……

  10. 导出

  11. 在Pr工程中新建素材箱，把XML这文件导入进来。

  12. 把XML文件拖入到时间轴上，ok

视频效果

拉长腿效果：

  1. 效果面板里找到“变换”

  2. 再到效果控件里找到“变换”

  3. 变换选择“矩形”的按钮
  4. 把整个下半身屏幕盖住
  5. 取消勾选等比缩放
  6. 拖动“缩放高度”数值
  7. 若发现有瑕疵：
  8. 在效果控件的变换里调整位置

运动打马赛克效果：

  1. 效果面板搜索”马赛克“

  2. 把效果拖到时间轴上

  3. 调整马赛克块值
  4. 画一个圆形，拖到人物脸上
  5. 点击蒙版上的小三角，进行跟踪

玻璃罩划过效果

  1. 视频复制一层到第二图层（原来的上面）

  2. 矩形工具（长按钢笔工具）画玻璃，在第二图层上面

  3. 效果面板搜索”轨道“

  4. 将”轨道遮罩键“拖动到第二轨道

  5. 在效果控件中改”轨道遮罩键“的”遮罩“选为矩形所在的图层

  6. 可以再添加”斜面Alapha“效果
