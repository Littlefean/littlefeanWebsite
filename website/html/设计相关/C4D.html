操作

视角操作

Alt + 鼠标左键 = 旋转视角

Alt + 鼠标中键 = 视角移动

Alt + 鼠标右键 = 放大缩小视图

鼠标中间 = 切换视图 三视图模式（想单独看某一个视图就单独用鼠标中间点视图，再点一下就回来）

这四个分别在视图右上角4个按钮代替

F1 F2 F3 F4 = 分别在这些窗口之间切换

渲染视图操作

舞台左上角---显示

​ 可以选择：光影着色或者线条

打开摄像机：

​ 舞台左上角---过滤---摄像机

物体操作

复制：

  1. ​ Alt 键拖动

  2. ​ 在右上角文件面板直接ctrlCV

移动旋转：

​ 工具栏前三个黄色图标

​ XYZ取消掉实现移动方向限制

​ 全局坐标：使用全局坐标移动被旋转了的物体

渲染

工具栏：

​ 开始渲染

​ 渲染到图片查看器

​ 渲染设置

物体面板

查看和隐藏

​ 点击物体图层右边的√，显示和隐藏

​ 左边的两个点也是一样

​ 上边的点直接显示隐藏

​ 下面的点是渲染的时候隐藏

​ 按住 Alt Shift 点小圆点，可以同时将两个隐藏，往下抹可以批量操作

平滑标签

​ √右边的两个橘色球标志

​ 作用是让曲面更加圆滑

添加其它标签

​ 小球，地面：刚体，碰撞体

图层：

​ √左边的东西，可以将物体归类到图层里方便管理
