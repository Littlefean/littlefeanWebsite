基础

基本操作

getwd() 查看当前路径

默认文件路径是用户文件夹下有一个 Document / R

更改默认路径：setwd(dir=".....绝对路径....")

查看当前文件夹下文件：list.files()

列出操作历史记录： history(历史记录条数)

清空屏幕：ctrl L

保存工作空间： save.image()

退出： q()

使用帮助

  1. help.start()  # 会给出你一个关于R的帮助网站
  2. help(sum)  # 列出一个函数详细的信息
  3. ?plot  # 可以直接用问号表示
  4. args(plot)  # 输出一个函数的参数格式

一些示例图

  1. example("hist")
  3. demo(graphics)  # 列出很多绘制的图
  4. vignette()

如果明明有这个包，却help不到这个包的提示信息，是因为还没有载入这个包

这个时候就可以用两个问号来看

  1. ??qplot

本地搜索，模糊词汇匹配

  1. help.search("heatmap")

列出所有关于xxx的内容

  1. apropos("sum")
  2. apropos("sum", mod="function")  # 只列出函数有关的内容

安装和更新

安装指令：

  1. install.packages()
  2. # 首次使用该指令需要选择镜像站点，会弹出一个框，找到China选了就行
  4. # 安装vcd包
  5. install.packages("vcd")
  7. # 同时安装多个包
  8. install.packages(c("AER", "ca"))

显示当前的安装包

  1. library()

显示安装包的路径

  1. .libPaths()

更新安装的软件包

  1. update.packages()

使用

显示包的帮助文档

  1. help(package="xxx")
  2. library(help="xxx")

列出包中所使用的函数

  1. ls("package: vcd")

删除包

  1. remove.package("vcd")

R包的批量移植

  1. installed.packages()  # 列出已经安装的R包
  2. installed.packages()[,1]  # 只访问第一列
  4. Rpack <- installed.packages()[,1]
  5. save(Rpack, file="Rpack.Rdata")
  7. # 去另一台设备上打开这个保存了的文件
  8. for (i in Rpack) install.packages(i)

数据类型汇总

  1. 数值型
  2. 字符串
  3. 逻辑型 TRUE，FALSE， T， F

数列（向量）

可以理解为一维数组

  1. x <- c(1, 2, 3, 4, 5)  # 把数列12345赋值给x
快捷生成
  1. c(1:100)  # [1, 100]
  2. seq(from=1, to=100)
  3. seq(from=1, to=100, by=2)  # 隔一个蹦一个的
  4. seq(from=1, to=100, length.out=10)  # 限制输出的个数是10个 等差的
  6. rep(2, 5)  # 2 2 2 2 2  重复5次2
类型问题

向量里面的类型必须是统一的类型，如果里面但凡有一个字符串类型，那么整个里面其他的数字都变成字符串类型的了

向量运算

向量 * n ： 把这个向量里面的每一个元素都 * n

向量 + 向量： 把两个向量对应元素相加

向量[条件表达式]：输出符合条件的向量里面的内容

向量编程是R语言的精髓所在

  1. x <- x + 1  # 每个元素+1
  3. x > 5  # 对每个元素都比较一遍是否大于5，返回一个元素为布尔值的
  5. x == y  # 遍历每一个元素，看他们对应的元素是否相等
向量索引
  1. length(x) # 向量里面元素的数量
  2. x[1]  # 第一个值，不是从0开始！
  3. x[-1]  # 除了第一个元素的其他所有元素
  4. x[c(4:18)]  # 输出第四个到第十八个元素
  5. x[c(11, 11, 23, 34, 6)]  # 直接输出对应位置的元素，这个里面不能既有正数又有负数
  6. x[c(T,F,T,F)]  # 只把逻辑为真的输出出来
  7. x[c(T)]  # 所有的都是T了
  8. x[c(T, F)] # 会按照顺序进行判断，相当于 TFTFTFTF……
  9. # 如果 TF这个一个循环里比x的长度还长，后面不够的就会成缺失值NA
  10. y[y>5]  # 判断是否符合条件
  11. y[y>5 && y<9]
字符串向量
  1. z <- c("one", "two", "three", "four", "five")
  2. "one" %in% z  # 是否包含在内的运算符判断

判断，筛选

  1. z[z %in% c("one", "two")]
  2. # 输出z中符合逻辑表达式的元素
  4. z %in% c("one", "two")
  5. # 把z向量自动转化成了布尔型向量，每一个元素表示该位置上原来的元素是否符合逻辑表达式的内容

造字典

  1. names(y) # NULL
  2. names(y) <- c("a", "b", "c"...........)
  4. y
  5. # a b c d ....
  6. # 1 2 3 4 ....
  8. y["a"]  # 1
增加

扩展

  1. # 可以通过索引直接扩展
  3. x[101] = 101
  5. # 这时候原本有100个元素的向量就变成101个了
  6. # 如果中间有空隙，那么中间的元素就是NA了

插入

  1. # 在某个下标位置之后插入，下标是从1开始的
  2. append(x= v, values = 99, after = 5)  # 向量，插入的值，插入的位置
删除
  1. y <- y[-c(1:3)]  # 删除了1到3位置上的元素
查询
  1. max(x)  # 查找数列的最大值
  3. which.max(x)  # 返回元素的最大下标
  5. [y, l] = max(x)  # 用两个变量 最大值，最大下标接收max的结果

矩阵

  1. x <- c(1, 1, 1, 2, 2, 2, 3, 3, 3, 4, 5, 4)
  2. # 把数列转化成矩阵
  3. # 参数 （被转化的数列，转化后有多少行，转化后有多少列）
  4. y <- matrix(x, nr=3, nc=4)  # 行3列4
  6. # 
  7. # apply(矩阵, MARGIN=1（按行），函数)
  9. apply(y, MARGIN=1, max)  # 查找每一行的最大值
  10. > x<-c(1, 9, 3, 8, 6, 6, 4, 2, 7, 2, 5, 1)
  11. > y<-matrix(x, nr=3, nc=4)
  12. > y
  13.      [,1] [,2] [,3] [,4]
  14. [1,]    1    8    4    2
  15. [2,]    9    6    2    5
  16. [3,]    3    6    7    1
  17. > apply(y, MARGIN=1, max)
  18. [1] 8 9 7
  19. # 每一行的最大值找到了
  21. # 计算矩阵每一列的均值
  22. apply(y, MARGIN=2, mean)
  23. # 默认按列填充， byrow=TRUE  ( 按行填充 )
  25. # 求数列的中位数和平均数
  26. mean(x)
  27. median(x)
  29. # 计算矩阵的方差和标准差
  30. apply(y,MARGIN=2,var)
  31. apply(y,MARGIN=2,sd)
