线性规划基础

  1. 一个线性规划中只能含有一个目标函数，如果超过了两个，就要想办法整合成一个
  2. 求目标函数的最大值和最小值用 max = …… min = …… 表示
  3. 线性函数和非线性函数的本质区别是：目标函数是否是线性的，其余的都一致
  4. 非线性函数求解十分困难，基本得不到全局最优解

矩阵工厂

生产一维矩阵

  1. sets:
  2. factory /1..6/ : a, b;
  3. plant /1..3/ : x, y;
  4. endsets
  6. a(1) = 100;
  7. !可以修改一个集合里的一个值;

结果：

  1. A = (100, 2, 3, 4, 5, 6)
  2. B = (1, 2, 3, 4, 5, 6)
  3. X = (1, 2, 3)
  4. Y = (1, 2, 3)
  1. factory和plant都是两家不同的工厂
  2. /1..6/ 说明这个工厂生产 1行6列的矩阵
  3. 冒号后面的 量都接收了工厂生产出来的矩阵
  4. 工厂名可以随便起
  5. lingo不是一行一行读代码的，所以要用sets和endsets包围起来
  6. 矩阵完成后，工厂和矩阵就会脱离联系

矩阵使用的时候要付初始值

  1. sets:
  2. f /1..6/ : a;
  3. endsets
  5. data:
  6. a = 1, 2, 3, 4, 5, 6;
  7. enddata

注意：

  1. 不是每个矩阵都要赋初始值，有些矩阵就是我们要求解的变量
  2. 矩阵赋初始值的时候不能多也不能漏
  3. 赋值可以是整数，也可以是小数

循环

求 a1x1, a2x2, a3x3, ... a5x5 的最大值

  1. sets:
  2. gc /1..5/ : a, x;
  3. endsets
  5. data:
  6. a = 1, 2, 3, 4, 5;
  7. enddata
  9. max = s;
  10. @for( gc(i) : s = a(i) * x(i) );

for循环里先写循环范围，和循环变量，用矩阵工厂表示范围，用自定义i来表示变量

  1. @for( 矩阵工厂(循环变量): 语句、方程 );

求和

  1. sets:
  2. gc /1..5/ : a, x;
  3. endsets
  5. data:
  6. a = 1, 2, 3, 4, 5;
  7. enddata
  9. max = s;
  10. @for( gc(i) : s = a(i) * x(i) );
  11. @sum( gc(i) : x(i)) = 5000;

上面的求和语句表示 把 x里面的每一个数求和，这个求和的结果必须等于 5000作为一个约束条件

  1. sum求和里面不可以阔起来完整的求和语句
  2. 写好求和语句之后再加约束条件

工厂合并 —— 可以生产二维矩阵

  1. sets:
  2. factory /1..6/ : a;
  3. plan /1..8/: d;
  4. copperation(factory, plan): c, x;
  5. endsets

c, x 就是 6行8列的矩阵

  1. 矩阵名称(一个行矩阵工厂名, 另一个行矩阵工厂名): 获得矩阵的变量, 获得矩阵的变量, ...;

第一个矩阵工厂决定了合并矩阵的行数，第二个决定列数

二维工厂赋值

  1. data:
  2. c = 1, 2, 3, 4, 5
  3. 4, 5, 6, 7, 8
  4. 1, 1, 1, 1, 1
  5. 4, 5, 6, 7, 9;
  6. enddata

写法注意：最后以分号结尾，且只有最后一行有分号

二维矩阵对角线求和

  1. min = @sum( coo(i, j): c(i, j) * x(i, j) );

条件语句

一般只在分段函数里用到

  1. @free(x);
  2. @free(y);
  3. x = -10;
  4. y = @if(x#ge#0, x + 10, x - 10);
  1. @if(条件表达式， 成立的语句、式子，不成立的语句、式子)

注意：

  1. 出现频率十分之低
  2. 若有if语句，必须带一个else

内置函数

变量定界函数

  1. @bin(x); 限制x只能取0，1
  2. @gin(x); 限制x为整数
  3. @bnd(x); 限制 a ≤ x ≤ b，推荐直接替换两个约束条件

数学函数

  1. @sin(x)
  2. @cos(x)
  3. @tan(x)
  4. @log(x) 返回x自然对数的值，想用其他底就用换底公式
  5. @exp(x) e的x次方
  6. @abs(x)
  7. @sigh(x) 返回x的符号值，x ≥ 0 就是 1，x ＜ 0 就是 -1
  8. @floor(x) 向下取整
  9. @smax(x1, x2, x3, ...., xn) 返回其中的全部最大值
  10. @smin(x1, x2, x3, ....., xn) 返回其中的全部最小值
