Uniapp组件

视图容器

  • view

相当于div，但是包裹文字需要使用组件

属性

  • hover-class：按下时的样式
  • hover-start-time：按住多久出现点击态

  • hover-stay-time：松开后点击态出现的时间

  • scroll-view

用于区域滚动

属性

  • scroll-x、scroll-y：是否允许横向竖向滚动
  • upper-threshold、lower-threshold：距离左上角、右下角多远的时候触发scrolltoupper 、scrolltolower 事件
  • scroll-top、scroll-left：竖向、横向滚动条的位置
  • show-scrollbar：是否出现滚动条
  • refresher-enabled：开启自定义下拉菜单刷新
  • refresher-background：下拉刷新的背景颜色

  • enable-flex：是否启用flex布局

  • swiper

滑块视图容器，像轮播图，整个一个屏幕的切换

属性

  • indicator-dots：是否显示面板指示点
  • indicator-color：指示点颜色
  • indicator-active-color：当前选中的指示点颜色
  • autoplay：是否自动切换
  • current：当前所在滑块的 index
  • interval：自动切换时间间隔
  • circular：是否播放到末尾后重新回到开头
  1.   <swiper class="swiper" :indicator-dots="indicatorDots" :autoplay="autoplay" :interval="interval" :duration="duration">
  2.       <swiper-item>
  3.           <view class="swiper-item uni-bg-red">A</view>
  4.       </swiper-item>
  5.       <swiper-item>
  6.           <view class="swiper-item uni-bg-green">B</view>
  7.       </swiper-item>
  8.       <swiper-item>
  9.           <view class="swiper-item uni-bg-blue">C</view>
  10.       </swiper-item>
  11.   </swiper>
  12.   
  • match-media

类似媒体查询

  1.   <view>
  2.       <match-media :min-width="375" :max-width="800" >
  3.           <view>当页面最小宽度 375px， 页面宽度最大 800px 时显示</view>
  4.       </match-media>
  5.   
  6.       <match-media :min-height="400" :orientation="landscape">
  7.           <view>当页面高度不小于 400px 且屏幕方向为纵向时展示这里</view>
  8.       </match-media>
  9.   </view>
  10.   
  • movable-area、movable-view

必须area是父，view是直接子节点

  • cover-view

覆盖在原生组件上的文本视图

  1.   <video class="video" src="……">
  2.       <cover-view class="controls-title">简单的自定义 controls</cover-view>
  3.       <cover-image @click="play" src="……"></cover-image>
  4.       <cover-image @click="pause" src="……"></cover-image>
  5.   </video>
  6.   
  • cover-image

覆盖在原生组件上的图片视图，可覆盖的原生组件同cover-view，支持嵌套在cover-view里

基础内容

  • icon 图标

  • type：icon的类型

  • size：icon的大小，单位px
  • color：icon的颜色，同css的color
  1.   <icon :type="value" size="26"/>
  2.   

  • text 文本组件

  • selectable、user-select：文本是否可选

  • space：显示连续空格
    • ensp：中文字符空格一半大小
    • emsp：中文字符空格大小
    • nbsp：根据字体设置的空格大小
  • decode：是否解码

注意：

  • <text> 组件内只支持嵌套 <text>
  • app-nvue下，只有<text>才能包裹文本内容
  • 支持 \n 方式换行
  1.   <text>{{text}}</text>
  2.   

  • rich-text 富文本

  • nodes：节点列表 / HTML String

  • space：显示连续空格

  • selectable：富文本是否可以长按选中，可用于复制，粘贴等场景

  • progress 进度条

  • percent：百分比

  • show-info：是否在右侧显示百分比
  • border-radius：圆角大小
  • font-size：右侧百分比字体大小
  • stroke-width：进度条线的宽度，单位px
  • activeColor：已选择的进度条的颜色
  • backgroundColor：未选择的进度条的颜色
  • duration：进度增加1%所需毫秒数
  • @activeend：动画完成事件

表单组件

  • button 按钮

  • size：按钮的大小

  • type：按钮的样式类型
  • plain：是否镂空，背景色透明
  • disabled：是否禁用
  • form-type：用于 <form> 组件，点击分别会触发 <form> 组件的 submit/reset 事件

  • 按下的样式变化，大部分和view相同

  • checkbox-group、checkbox 多项选择器

  1.   <checkbox-group @change="checkboxChange">
  2.       <label class="uni-list-cell uni-list-cell-pd" v-for="item in items" :key="item.value">
  3.           <view>
  4.               <checkbox :value="item.value" :checked="item.checked" />
  5.           </view>
  6.           <view>{{item.name}}</view>
  7.       </label>
  8.   </checkbox-group>
  9.   
  • editor 富文本编辑器

可以对图片、文字格式进行编辑和混排。

  • form 表单
  1.   <view>
  2.       <form @submit="formSubmit" @reset="formReset">
  3.           <view class="uni-form-item uni-column">
  4.               <view class="title">radio</view>
  5.               <radio-group name="radio">
  6.                   <label>
  7.                       <radio value="radio1" /><text>选项一</text>
  8.                   </label>
  9.                   <label>
  10.                       <radio value="radio2" /><text>选项二</text>
  11.                   </label>
  12.               </radio-group>
  13.           </view>
  14.       </form>
  15.   </view>
  16.   

  • input 输入框

  • value：输入框的初始内容

  • type：input 的类型
    • text：文本输入键盘
    • number：数字输入键盘
    • idcard：身份证输入键盘
    • digit：带小数点的数字键盘
  • password：是否是密码类型
  • placeholder：输入框为空时占位符
  • disabled：是否禁用
  • maxlength：最大输入长度，-1不限制长度
  • focus：获取焦点
  • cursor：指定focus时的光标位置
  • @input：当键盘输入时，触发input事件
  • @focus：输入框聚焦时触发
  • @blur：输入框失去焦点时触发
  • @confirm：点击完成按钮时触发

  • @keyboardheightchange：键盘高度发生变化的时候触发此事件

  • label 改进表单组件的可用性

  • picker 从底部弹起的滚动选择器

  • @change：value 改变时触发 change 事件

  • @columnchange：某一列的值改变时触发 columnchange 事件

  • @cancel：取消选择时触发

  • picker-view 嵌入页面的滚动选择器

注意：其中只可放置 <picker-view-column/> 组件，其他节点不会显示

  1.   <picker-view v-if="visible" :indicator-style="indicatorStyle" :value="value" @change="bindChange" class="picker-view">
  2.       <picker-view-column>
  3.        <view class="item" v-for="(item,index) in years" :key="index">{{item}}年</view>
  4.       </picker-view-column>
  5.       <picker-view-column>
  6.        <view class="item" v-for="(item,index) in months" :key="index">{{item}}月</view>
  7.       </picker-view-column>
  8.       <picker-view-column>
  9.        <view class="item" v-for="(item,index) in days" :key="index">{{item}}日</view>
  10.       </picker-view-column>
  11.   </picker-view>
  12.   

  • radio-group、radio 单选

  • value：<radio> 标识。当该 <radio> 选中时，<radio-group> 的 change 事件会携带 <radio> 的 value

  • checked：当前是否选中
  • disabled：是否禁用

  • color：radio的颜色，同css的color

  • slider 滑动选择器

  • min、max：最小值最大值

  • step：步长，取值必须大于 0，并且可被(max - min)整除
  • value：当前值
  • activeColor：滑块左侧已选择部分的线条颜色
  • backgroundColor：滑块右侧背景条的颜色
  • block-size：滑块的大小，（12 - 28）
  • block-color：滑块的颜色

  • show-value：是否显示当前 value

  • switch 开关选择器

  • checked：是否选中

  • disabled：是否禁用
  • type：样式，有效值：switch, checkbox
  • @change：checked 改变时触发 change 事件

  • color：switch 的颜色

  • textarea 多行输入框

  • value：输入框的内容

  • placeholder：输入框为空时占位符
  • maxlength：最大输入长度，设置为 -1 的时候不限制最大长度
  • focus：获取焦点
  • auto-height：是否自动增高

跳转

  • navigator 类似 a标签
  • custom-tab-bar 自定义tabBar组件
  • direction：选项的排列方向
  • show-icon：是否显示icon
  • selected：选中的tabBar选项索引值
  • onTabItemTap：点击事件，参数为Object

媒体组件

  • audio 音频

  • camera 相机

  • image 图片

  • src：图片资源地址

  • mode：图片裁剪、缩放的模式
  • lazy-load：图片懒加载。
  • fade-show：图片显示动画效果

注意：

  1. <image> 组件默认宽度 300px、高度 225px；`app-nvue平台，暂时默认为屏幕宽度

  2. src 仅支持相对路径、绝对路径，支持 base64 码；

  3. video 视频播放组件

画布

  • canvas
  • type：指定 canvas 类型
  • canvas-id：canvas 组件的唯一标识符
  • disable-scroll：当在 canvas 中移动时且有绑定手势事件时，禁止屏幕滚动以及下拉刷新

webview

  • web-view

web 浏览器组件，可以用来承载网页的容器，会自动铺满整个页面（nvue 使用需要手动指定宽高）

导航类组件

  • navigation-bar 用于指定导航栏的一些属性
  • title：导航条标题
  • title-icon：标题图标
  • title-icon-radius：标题图标圆角
  • subtitle-text：副标题文字内容
  • subtitle-size：副标题文字字体大小
  • subtitle-color：副标题文字颜色
  • subtitle-overflow：标题文字超出显示区域时处理方式
  • title-align：对齐方式
  • front-color：导航条前景颜色值
  • background-color：导航条背景颜色值
  • custom-tab-bar 自定义tabBar组件
  • direction：选项的排列方向 可选值：horizontal，vertical
  • show-icon：是否显示icon
  • selected：选中的tabBar选项索引值
  • onTabItemTap：点击事件，参数为Object
    • index：被点击tabItem的序号，从0开始
    • pagePath：被点击tabItem的页面路径
    • text：被点击tabItem的按钮文字

页面属性配置节点

  • page-meta

page-meta对pages.json有一定替代作用，可以让页面的配置和页面内容代码写在一个vue文件中。

注意：page-meta只能是页面内的第一个节点

  • background-text-style：下拉背景字体
  • background-color：窗口的背景色
  • background-color-top：顶部窗口的背景色
  • background-color-bottom：底部窗口的背景色
  • scroll-top：滚动位置
  • scroll-duration：滚动动画时长

  2.     <page-meta
  3.       :background-text-style="bgTextStyle"
  4.       :background-color="bgColor"
  5.       :background-color-top="bgColorTop"
  6.       :background-color-bottom="bgColorBottom"
  7.       :scroll-top="scrollTop"
  8.       page-style="color: green"
  9.       root-font-size="16px"
