Vue

使用和部署

  1. <script src="vue.js"></script>

引入之后会暴露一个全局的变量： Vue， 通过函数进行操作 Vue()

第二种部署

  1. <script src="https://cdn.jsdelivr.net/npm/vue"></script>

声明式渲染

创建第一个 vue.js 的应用

  1. <div id="app">
  2.   {{ message }}
  3. </div>
  4. <script>
  5.     // Vue 需要接收一个对象，对象有两个重要的参数     var app = new Vue({
  6.         // 代表element       el: '#app',
  7.         // 代表里面的信息       data: {
  8.         message: 'Hello Vue!'
  9.       }
  10.     })
  11. </script>

生命周期

每一个vue对象创建的时候都要经过一些列初始化过程

不要使用箭头函数

  1. var vm = new Vue({
  2.     el: '#example',
  3.     data: {
  4.         msg: 'hi vue'
  5.     },
  6.     // 实例化之后，数据观测 和 事件配置 之前     // 整个页面创建之前     beforeCreate: function(){
  7.         //      },
  8.     // 在实例完全创建好了之后     // 但是挂载阶段还没开始 $el 属性还不可见     created: function() {
  9.      // },
  10.     // 挂在之前被调用，相关的渲染函数首次被调用     beforeMount: function() {
  11.         //     },
  12.     // 挂载成功，所有的节点被vue的节点替换     mounted: function() {
  13.         //     },
  14.     // 数据变化之前     beforeUpdate: function() {
  15.         //     },
  16.     // 数据更新完毕了，DOM结构也已经更新了     update: function() {
  17.         //     }
  18.     // 当vue的属性发生了变化之后，会相继执行一次 beforeUpdate 和 update 两个函数 })

指令

条件渲染

v-if="变量"

  1. <div id="app">
  2.     <p v-if="seen"> <!-- seen 是data属性里的一个布尔值 -->
  3.         you see this massage is my
  4.     </p>
  5. </div>
  6. <script>
  7. var vm = new Vue({
  8.         el: '#app',
  9.         data: {
  10.             seen: true
  11.         }
  12.     });
  13. </script>

v-else-if

  1. <div v-if="布尔变量1">
  2.     123
  3. </div>
  4. <div v-else-if="布尔变量2">
  5.     123456
  6. </div>
  7. <div else>
  8.     123456789
  9. </div>

相当于

  1. if 布尔变量1:
  2.     print(123)
  3. elif 布尔变量2:
  4.     print(123456)
  5. else:
  6.     print(123456789)

v-show

  1. <h1 v-show="true">
  2.     ....
  3. </h1>

如果是 false，它的标签还会存在到 html的标签上，但是样式 display 会变成 none

如果是前面的条件渲染方式，则根本不会出现在html的标签里面，直接就没了

v-if是真正的条件渲染

一般来说 v-if 有更高的切换开销，v-show 有更高的初始化开销，就看那个频繁，用相反的那个就可以

修饰符

用 @click 属性可以为元素绑定点击事件

  1. <div @click="clickfunc1">
  2.     <div @click="clickfunc2">
  3.     click me        
  4.     </div>
  5. </div>
  6. <script>
  7.     var vm = new Vue({
  8.         el: '#app',
  9.         data: {
  10.             seen: false;
  11.             url: "https......."
  12.         },
  13.         // 点击方法要写到 methods 里面         methods: {
  14.             clickfunc1: function() {
  15.                 // 点击事件的函数             }
  16.         }
  17.     });
  18. </script>

注意：此处点击之后会执行两个方法，最内层先执行，然后一直到父级也执行，为了只触发一个函数，可以添加 stop

  1. <div @click="clickfunc1">
  2.     <div @click.stop="clickfunc2">
  3.     click me        
  4.     </div>
  5. </div>

常见的事件修饰符

  • .stop
  • .prevent
  • .caputre
  • .self
  • .once
  • .passive

  3. </div>

  3. </div>

事件处理

v-on 监听 DOM 事件，并在触发时运行一些JavaScript代码

  1. <标签 v-on:事件名="所执行的代码"></标签>
  1. <标签 v-on:事件名="函数名(参数1, 参数2...)"></标签>
  1. <button v-on:click="counter += 1">
  2.     数值 : {{ counter }}
  3. </button>

注意：如果是函数，函数的声明要在vue的methords属性里

  1. <div id="app">
  2. <button v-on:click="func('abc', $event)">
  3.         
  4.     </button>
  5. </div>
  6. <script>
  7.     var vm = new Vue({
  8.         el: '#app',
  9.         data: {
  10.             seen: false;
  11.             url: "https......."
  12.         },
  13.         // 点击方法要写到 methods 里面         methods: {
  14.             func: function(str, e){
  15.                 alert(str);
  16.                 console.log(e)
  17.             }
  18.         }
  19.     });
  20. </script>

写 $event 可以传入所有事件

常用事件

  • click：点击
  • dblclick：双击

表单输入绑定

v-model 实现双向绑定

输入框

  1. <input v-model="vue变量">
  2. <p>
  3.     {{vue变量}}
  4. </p>

实现了输入什么值，p标签里面的内容也就跟着进行绑定的过程

单选框

  1. <div >
  2.     <input type="radio" value="1" v-model="vue变量">
  3.     <input type="radio" value="2" v-model="vue变量">
  4.     <input type="radio" value="3" v-model="vue变量">
  5. </div>
  6. <p>
  7.     {{vue变量}}
  8. </p>

p标签里展示出来的 是单选的那一项对应的 value值

多选框

  1. <div >
  2.     <input type="checkbox" value="1" v-model="vue数组变量">
  3.     <input type="checkbox" value="2" v-model="vue数组变量">
  4.     <input type="checkbox" value="3" v-model="vue数组变量">
  5. </div>
  6. <p>
  7.     {{vue数组变量}}
  8. </p>
