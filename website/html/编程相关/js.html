          正则表达式字符

          注意：正则表达式里面不需要加引号，不管是数字类型还是字符类型

          边界符
          1. ^ ：表示匹配行首的文本（以谁开始）
          1.    var rg = /^abc/;
          2.    console.log(rg.test("abc")); //true    console.log(rg.test("sabc")); //false    
          1. $ ：表示匹配行尾的文本（以谁结束）
          1.    var rg = /^abc$/;
          2.    //精确匹配：以abc开头，并且以abc结尾，并且只有abc，abcabc不行    
          字符类
          1. [] ：只要字符串中包含内部的元素就返回true
          1.    var rg = /[abc]/;
          2.    
          3.    var rg = /^[abc]$/; //三选一，只有是a，或者b，或者c，才正确。    
          1. - ：范围字符
          1.    var rg = /^[a-z]$/; //只要是任意一个字母    var rg = /^[a-zA-Z0-9]$/; //只要是任意一个不区分大小的字母或数字    var rg = /^[a-zA-Z0-9_-]$/; //只要是任意一个不区分大小的字母或数字，下划线或者短横线    
          1. ^ ：中括号内的取反符号
          1.    var rg = /^[^a-zA-Z]$/;  //不能包含英文字母    
          量词符

          设定某个模式出现的次数

          1. * ：重复多少次
          1.    var rg = /^a*$/;  //可以任意次数a    
          1. + ：出现次数 >= 1
          1.    var rg = /^a+$/;  //出现次数大于等于1    
          1. ? ：出现 1 次或 0 次
          1.    var rg = /^a?$/;
          2.    
          1. {} ：重复 n 次
          1.    var rg = /^a{3}$/; //a出现3次    var rg = /^a{3,}$/; //大于等于3次    var rg = /^a{3,6}$/; //大于等于3次，小于等于6次    var rg = /^abc{3}$/; //abccc，优先级有问题，只是让c重复三次    

          注意：输入量词的时候中间不要有空格
