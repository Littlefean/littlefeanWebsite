定式

产生一个窗口

  1. from Tkinter import *   #引用Tk模块
  2. root = Tk()             #初始化Tk()
  3. root.mainloop()         #进入消息循环

几个常用属性

  1. from Tkinter import *
  2. root = Tk()
  3. root.title("hello world")
  4. root.geometry('200x100')                 #是x 不是*
  5. root.resizable(width=False, height=True) #宽不可变, 高可变,默认为True
  6. root.mainloop()

控件

每个控件最后要加上pack()否则控件是无法显示的

Label

说明：标签

  1. Label(根对象, [属性列表])

  • text　 要现实的文本

  • bg　　 背景颜色

  • font　 字体(颜色, 大小)

  • width　 控件宽度

  • height　控件高度

  1. from Tkinter import *
  2. root = Tk()
  3. root.title("hello world")
  4. root.geometry('300x200')
  5. l = Label(root, text="show", bg="green", font=("Arial", 12), width=5, height=2)
  6. l.pack(side=LEFT)  #这里的side可以赋值为LEFT  RTGHT TOP  BOTTOM
  7. root.mainloop()

特殊问题

关闭窗口后子线程依旧在运行

给线程添加一个代码

  1. t = threading.Thread(target=tick)
  2. t.setDaemon(True)  # 此行代码解决关闭窗口后程序没有结束的bug
  3. t.start()
