Lambda表达式

介绍

代替匿名内部类的写法

限制指定了一种做事情的方案，其实就是一个函数

标准格式

三部分组成

  1. 一个小括号 参数
  2. 一个箭头 传递参数
  3. 一个大括号 一段代码

( 参数列表 ) -> { 一段代码 };

解释说明格式：

()：没有参数就保留括号，有多个参数就以逗号隔开

->：把参数传递给方法体

{}：重写接口的抽象方法

省略格式

它可推到，可省略的，凡是根据上下文能够堆到出来的内容，都可以省略书写

可以省略的内容：

  1. 参数列表中参数的类型
  2. 括号本身（参数只有一个的时候）
  3. 大括号本身，return，分号，一起省略（大括号内部只有一个语句的时候）
