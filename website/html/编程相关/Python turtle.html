学习笔记 Python turtle

导入

import turtle

基本方法

窗口

turtle.setup(width,height,startx,starty) 设置窗口大小

turtle.mainloop() 结束绘制后将窗口固定

颜色

turtle.color('red', 'pink') 设置画笔颜色和填充颜色

turtle.pencolor() 画笔颜色

turtle.fillcolor() 填充颜色

turtle.begin_fill() 开始填充颜色

turtle.end_fill() 停止填充颜色

具体操作方法

绘制

turtle.pendown() 开始绘制

turtle.penup() 停止绘制

方向

turtle.seth(60) 设置方向

turtle.left(60) 左转60度

turtle.right(60) 右转60度

前进

turtle.fd(100) 前进100

turtle.forward(100) 前进100

turtle.goto(100, 200) 直接到某个地方

turtle.circle(100, 90) 圆弧线走，半径100， 圆心角90
