Random

choice

随机选择一个元素

  1. a = choice([True, False])

shuffle

洗牌函数，打乱列表的顺序

注意：该函数没有返回值

  1. b = [1, 2, 3, 4, 5, 6, 7, 8]
  2. shuffle(b)
  3. print(b)

sample

从列表中随机抽多个函数

  1. c = sample([1, 2, 3, 4, 5, 6, 7, 8], 3)
  2. print(c)

uniform

随机浮点数

注意：左右两个参数位置可以互换

  1. d = uniform(0, 1)
  2. print(d)
  3. d = uniform(1, 0)
  4. print(d)

triangular

随机浮点数（能够加权）

注意：第三个参数如果不写的话，会是默认前两个参数的中点，随机到的值会更接近第三个参数，而uniform是均匀分布

  1. for i in range(100):
  2.     u = uniform(1, 10000)
  3.     t = triangular(1, 10000, 10)
  4.     print(int(t), end="\t\t\t")
  5.     print(int(u))

对比triangluar和unifrom的区别：

  1. im = Image.new('RGB', (1000, 1000), (0, 0, 0))
  2. for i in range(100000):
  3.     t1 = triangular(1, 999, 1)
  4.     t2 = triangular(1, 999, 1)
  5.     im.putpixel((int(t1), int(t2)), (255, 255, 255))
  6.     pass
  7. im.save("randomTestTriangular.png")

  1. im = Image.new('RGB', (1000, 1000), (0, 0, 0))
  2. for i in range(100000):
  3.     u1 = uniform(1, 999)
  4.     u2 = uniform(1, 999)
  5.     im.putpixel((int(u1), int(u2)), (255, 255, 255))
  6. im.save("randomTestUniform.png")

注意：参数的条件是 alpha > 0 和 beta > 0。 返回值的范围介于 0 和 1 之间。

  1. a = betavariate(1, 1)

normalvariate

正太分布

第一个参数是平均值 μ，第二个参数是标准差 σ

  1. for i in range(100):
  2.     a = normalvariate(100, 1)
  3.     print(a)
  1. 平均值 μ 决定了数据所在的范围，
  2. 标准差 σ 表示数据的集中程度， σ 越小越集中

vonmisesvariate

冯·米塞斯分布

冯·米塞斯（von Mises）分布。 mu 是平均角度，以弧度表示，介于0和 2pi 之间，kappa 是浓度参数，必须大于或等于零。 如果 kappa 等于零，则该分布在 0 到 2pi 的范围内减小到均匀的随机角度。

paretovariate(alpha)

帕累托分布，alpha是形状参数

返回（1，+∞）的一个数，大数占极少数

alpha越大，贫富差距越小

weibullvariate(alpha, beta)

威布尔分布（看成一种只含有正半轴的正太分布）

alpha 是比例参数，beta 是形状参数。

alpha：这个数决定数据更容易生成在哪里，相当于mu

beta：这个数决定了数据的集中程度，越大越集中
