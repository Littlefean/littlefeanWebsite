泛型

介绍

方法返回值有要求既可以是数字类型，也可以是字符串类型的时候，可以用any，但是频繁使用any并不合理，这时候可以使用泛型

  1. function Hello<T>(arg:T):T {
  2. return arg;
  3. }
  4. // <T> 把当前的函数指定成一个类型 // arg:T 参数指定为一个泛型类型 // ):T 返回值类型也指定为一个泛型类型
  5. var output = Hello<string>("Hello ime");

泛型的好处：规定了所有的 T 都要是同一个类型

注意：不一定要用 T 来表示，只是一种习惯

应用

  1. function Hello<T>(num:T):T {
  2.     alert(num.length);  // 编译不通过，num没有length属性     return num;
  3. }
  1. function Hello<T>(str:T[]):T[]{
  2.     alert(str.length);
  3.     return str;
  4. }
  5. // 编译通过 // 说明泛型是根据类型来决定 自己所具有哪些属性的 var list:Array<stirng> = Hello<string>(["a", "b", "c"]);
  6. for (var i = 0; i < list.length; i++) {
  7.     alert(list[i]);
  8. }

泛型类型

  1. function Hello<T>(arg:T):T {
  2.     return arg;
  3. }
  4. var myHello:<K>(arg:K) => K = Hello;
  5. alert(myHello("hello"));

myHello的类型是 <K>(arg:K) => K 这样一个函数类型；

  1. function Hello<T>(arg:T):T {
  2.     return arg;
  3. }
  4. var myHello:{<T>(arg:T):T} = Hello;
  5. alert(myHello("Hello"));

myHello的类型是 <T>(arg:T):T 这样的函数类型；

利用接口

方式1

  1. interface Hello {
  2. <T>(arg:T):T;
  3. }
  4. function myHello<T>(arg:T):T {
  5.     return arg;
  6. } 
  7. var MH:Hello = myHello;
  8. alert(MH<string>("hello"));

方式2

  1. interface Hello<T> {
  2.     (arg:T):T;
  3. }
  4. function myHello<T>(arg:T):T {
  5.     return arg;
  6. }
  7. var mh:Hello<number> = myHello;
  8. alert(mg(100));
