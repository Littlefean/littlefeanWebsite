列表循环

wx:for

  1. <view wx:for="{{数组 或者 对象}}" wx:for-item="循环项的名称" wx:index="循环项索引">
  2. {{item}}
  3.     {{index}}
  4. </view>

wx:key

wx:key = "唯一的值" 用来提高列表的渲染性能

  1. 绑定一个普通的字符串的时候，这个字符串名称 肯定是 循环数组 中的 对象的唯一属性
  1.  data: {
  2.     msg: "iwen",
  3.     arr: [
  4.         {name: "a", age: 15},
  5.         {name: "b", age: 15},
  6.         {name: "c", age: 15},
  7.         {name: "d", age: 15},
  8.         {name: "e", age: 15},
  9.     ]
  10.   },
  1. <view wx:for="{{数组 或者 对象}}" wx:for-item="循环项的名称/对象的值" wx:index="循环项索引/对象属性" wx:id="name">
  2. {{item}}
  3.     {{index}}
  4. </view>

wx:key = "*this" 就表示 数组是一个普通的数组

  • 当出现 数组循环嵌套的时候，注意绑定的名称不要重名
  • 如果只有一层循环的话，item和index可以省略

block标签：被循环了之后会被自动去掉，外面的标签会被自动去掉

基础组件

常见组件

  1. view, text, rich-text, button, image, navigation, icon, swiper, radio, checkbox

  1. view 代替 div

  2. text 文本标签

  3. 文本

  4. 只能嵌套他自己
  5. 长按可以复制（只有这个有这个功能 属性：selectable

  6. 可以编码空格和回车 属性：decode 

  7. image

  8. 图片的默认宽高是 320px 240px

  9. mode 属性决定图片宽度高度属性如何和大小相适配

    • scaleToFill 默认值 不保持纵横比缩放图片
    • aspectFit 保持比例，保证长边完全显示
    • aspectFill 保持比例，只保证短边显示出来
    • widthFix 保持比例，宽度固定
    • left top bottom right 背景图片定位，不缩放图片

  10. 支持懒加载 添加属性 lazy-load

    就是当图片出现在视口上的时候，那就加载图片

  11. swiper 轮播图组件

  12. 必须搭配组件 swiper-item 组件使用

  13. swiper 存在默认高度，150px，高度不能默认撑开
  14. 图片存在默认宽度和高度，320*240
  15. 需要根据图片的宽度和高度来定 swiper 的宽度和高度

  16. 轮播属性

    1. autoplay 自动轮播
    2. interval 播放间隔 毫秒
    3. circular 是否衔接轮播
    4. indicator-dos 是否显示小圆点
    5. indicator-color 未选中小圆点的颜色
    6. indicator-active-color 选中小圆点的颜色

  17. navigator 导航栏组件

  18. 块级元素，默认换行
