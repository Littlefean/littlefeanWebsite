HTML 学习笔记

介绍

Hypro text markup langue

元素标签分类

块元素

特点：

  1. 自己独占一行
  2. 宽度高度内外边距都可以控制
  3. 宽度默认是父元素的100%
  4. 是一个容器，里面可以放行内或者块级元素

注意：

文字类元素里不能使用块级元素 如 p h1~h6

行内元素

特点：

  1. 一行显示多个
  2. 宽高直接设置无效
  3. 默认本身内容的宽度
  4. 只能容纳文本或其他行内元素

注意：

a标签里可以放块级元素

行内块元素

举例：img input td 同时具有行内元素和块元素的特点

特点：

  1. 和相邻行内块元素在一行上，但是它们之间会有空白间隙，一行可以显示多个
  2. 默认宽度就是它本身内容的宽度
  3. 高度宽度，行高，内外边距可控

html公式

输入公式结束后在公式末尾按TAB键

标签名>标签名 → Tab 快速嵌套

标签名*n → Tab 生成n个相同标签

标签名+标签名 → Tab 并列关系

div>(div+a)*10 小括号优先运算

div>span^div 使后面的跳出包围（可以多加几个^跳出多级）

div[title="jiji"] 中括号，表示元素属性

ul>li.menu$*5 变量，会用12345代替（多个$可以在前补零）

ul>li.item$@-*10 倒顺牌列，在$后加 @-

ul>li.item$@6*10 自定义起始值 @后加一个数字

div{text} 大括号将内容放入标签中

! 生成html初始结构

html:5 生成html5初始结构

link 生成链接代码 放在head里面

script 插入js代码块

script:src 引用js文件 放在任何位置都可以，建议放在最后

img 插入图片

a 空格，回车

ul>.menu 自动生成列表，里面的标签名可以省略

注意

  1. 不能空格，因为空格是停止
  2. 光标放在最后按Tab键
  3. 可以在任何位置键入缩写
