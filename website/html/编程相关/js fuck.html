Js Fuck

介绍

只用六种字符就可以实现js编程，非常艹蛋，所以叫jsFuck。

( ) [ ] + !

官网地址： http://www.jsfuck.com/

原理

  1. 元素前加 +：把这个东西解析成数字
  2. 某元素 + []：把这个东西转化成字符串，如果是变量名 + [] 会把变量的值转化成字符串
  3. 元素前加 !：转化成bool类型

六种符号优先级

  1. 纯圆括号 ： ()
  2. 成员访问： ...[...]
  3. 函数调用： ...(...)
  4. 后置递增： ...++
  5. 非： !...
  6. 前加号： +...
  7. 前置递增： ++...
  8. 加法： ...+...

基本元素对照表

  1. +[] == 0
  2. ![] == false
  3. !![] == true
  4. +!![] == 1
  6. !![]+[] == "true"
  8. (!![]+[])[+[]] == 't'  // ("true")[0]
  9. undefined   ==  [][[]]  // 对一个空数组取下标，但是取下标取的不对，所以返回 undefined NaN         ==  +[![]]  // 把一个数组 [false] 解析成数字，解析成了非数字 NaN 2           ==  !+[]+!+[]
  10. 10          ==  [+!+[]]+[+[]]
  12. Array       ==  []  // 默认的数组就是空数组 Number      ==  +[]  // 默认的数字类型是0 Boolean     ==  ![]  // 默认的布尔值就是假
  13. Function    =>  []["filter"]
  14. eval        =>  []["filter"]["constructor"]( CODE )()
  15. window      =>  []["filter"]["constructor"]("return this")()
