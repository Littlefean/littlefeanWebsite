Python 内置函数

complex : 复数

divmode() 返回商和余数

format() 与具体数据相关, 用于计算各种小数, 精算等.

  1. print(format(s, "^20"))  #剧中*
  2. print(format(s, "<20"))  #左对齐*
  3. print(format(s, ">20"))  #右对齐*
  4. print(format(3, 'b' )) # 二进制:11
  5. print(format(97, 'c' )) # 转换成unicode字符:a
  6. print(format(11, 'd' )) # ⼗进制:11
  7. print(format(11, 'o' )) # 八进制:13
  8. print(format(11, 'x' )) # 十六进制(⼩写字母):b
  9. print(format(11, 'X' )) # 十六进制(大写字母):B
  10. print(format(11, 'n' )) # 和d⼀样:11
  • ord() 输入字符找带字符编码的位置：字符→数字
  • chr() 输入位置数字找出对应的字符：数字→字符
  • ascii() 是ascii码中的返回该值 不是就返回u

frozenset() 创建一个冻结的集合，冻结的集合不能进行添加和删除操作。

  • eval() 执行字符串类型的代码. 并返回最终结果
  • exec() 执行字符串类型的代码
  • compile() 将字符串类型的代码编码. 代码对象能够通过exec语句来执行或者eval()进行求值

内存相关

  • hash() 获取哈希值
  • id() 获取内存地址
