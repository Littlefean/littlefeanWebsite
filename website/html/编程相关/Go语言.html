Golang 学习笔记

环境搭建

新建GOPATH环境变量文件夹，把之前默认的user/用户名/go改成自己设定的路径

在文件夹里新建三个文件夹：bin（编译出来得二进制文件）、pkg（打包）、src（源代码文件）

查看go相关的环境变量

cmd输入 go env

在Vscode中配置Go开发环境

  1. 您只需通过 Go 命令进行简单设置
  1.    go env -w GOPROXY=https://goproxy.io,direct

  3.    # 还可以设置不走 proxy 的私有仓库或组，多个用逗号相隔（可选）    go env -w GOPRIVATE=*.corp.example.com,github.com/org_private
  4.

开始配置代码快捷键

参考 https://www.liwenzhou.com/posts/Go/00_go_in_vscode/

  1. ctrl+shift+p 之后输入 >snippets

  2. 选择go，进入 go.json

  3. 用法

  1.    “这里放个名字”:{
  2.        "prefix": "这个是快捷键",
  3.        "body": "这里是按快捷键插入的代码片段",
  4.        "description": "这里放提示信息的描述"
  5.    }
  6.    

其中$0表示最终光标提留的位置。 举个例子，我这里创建了两个快捷方式，一个是输入pln就会在编辑器中插入fmt.Println()代码；输入plf，就会插入fmt.Printf("")代码。

  1.    {
  2.        // 。。。
  3.        // 。。。
  4.        // 。。。它上面给的注释
  5.     "println":{
  6.     "prefix": "pln",
  7.     "body":"fmt.Println($0)",
  8.     "description": "println"
  9.     },
  10.     "printf":{
  11.     "prefix": "plf",
  12.     "body": "fmt.Printf(\"$0\")",
  13.     "description": "printf"
  14.     }
  15.    }
  16.    
  1. 若出现 “fmt” 引用失败，检查命令行输入go env GOPATH 和 GOROOT，系统环境变量看GOPATH 和 GOROOT

格式解释

package

每个源文件都以一条package声明语句开始，

package main, 表示该文件属于哪个包，如果是main包，会被编译成可执行文件

紧跟着一系列导入（import）的包，之后是存储在这个文件里的程序语句。

import

必须跟在文件的package声明之后

注意要用双引号

格式

不需要在语句或者声明的末尾添加分号

函数的左括号{必须和func函数声明在同一行上, 且位于末尾，不能独占一行

命名

  • 名字的开头字母的大小写决定了名字在包外的可见性

  • 变量名尽量短，声明周期长的可以长，变量用小驼峰

标识符和关键字

标识符

字母、数字、下划线组成，和python规则一样

关键字

25个关键字，37个保留字，保留字是不推荐你使用的字

变量

必须先声明再使用

声名了之后尽量使用，不使用编辑器会提醒

声明

  1. // 完整 var 变量名 类型 = 数值或表达式
  2. // 缺省 var 变量名 = 数值或表达式
  3. // 缺少表达式 var 变量名 类型
  5. // 批量声明变量 var (
  6.      // 默认空值 name string // ""     age int     // 0     isOk bool   // false )

根据表达式自动断类型

  • 数值型对应0

  • 布尔型对应false

  • 字符串类型对应的零值是空字符串

  • 接口或引用类型（包括slice、指针、map、chan和函数）变量对应的零值是nil

  • 数组或结构体等聚合类型对应的零值是每个元素或字段都是对应该类型的零值。

  1. // 初始化多个变量 var i, j, k int
  2. var b, f, s = true, 2.3, "four" 

函数赋值声明变量

      简短声明

      例如：i := 100

      注意：

      • 不能重复声明
      • 只能在函数里使用
      • 声明必须包含新变量

      匿名变量

      在多重赋值的时候，忽略某个值，用于占位，用 _ 表示

      它不占用命名空间和内存，所以匿名变量之间不会重复命名
