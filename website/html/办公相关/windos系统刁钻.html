如何使用键盘

打开搜索功能

win + Q （直接按win键然后输入内容搜索）

打开Windows设置中心

win + i

打开文件资源管理器

win + E

打开我的电脑

  • win + R cmd内输入D:即可打开D盘

  • cmd内输入 cd 文件名 进入某一个文件夹

启动触摸板

  • ​ win → ↓→设置→输入触摸板→启用触摸板

开启笔记本功能

  • win + R 输入 notepad

我的电脑翻阅基本操作

  • Alt + ↑ 返回到上一级文件夹

  • Alt + ← 返回上一次打开的文件夹

  • Alt + Enter 查看属性

  • Alt + 双击 查看属性

  • Shift + Delete 永久删除文件

  • Alt → 空格 → C 关闭当前窗口

  • Alt → V 打开视图菜单

  • F2 选中状态的文件夹改名

  • Ctrl + Shift + N 创建文件夹
  • Ctrl + N 复制当前窗口
  • Ctrl + Y 取消撤销

浏览器操作

  • F3 查找关键字 = ctrl F

  • F5 刷新

  • F4 打开曾经输入的网址

  • F11全屏

  • F12 开发者模式

  • Ctrl + Shit + T 恢复误删掉的网页

  • Ctrl + F5 在IE浏览器中强行刷新

  • Ctrl + W 关闭标签

  • Ctrl + T　新建标签

  • Ctrl + D 收藏网页

桌面图标

设置桌面图标

桌面右键→个性化→主题→相关的设置→桌面图标设置

快速隐藏桌面图标

桌面右键→V→D

截屏

  1. PriScSysRq 这是所有电脑都自带的全屏截图 保存到粘贴板

  2. win + PriScSysRq 这是在上一个基础上能够把你的截图保存到电脑里，保存在C:/用户/*你的用户名/图片/Secreenshots里

  3. Alt + PriScSysRq 这是只截图当前窗口保存到粘贴板

  4. win + Alt + PriScSysRq 这是电脑截图只截图当前窗口并保存到电脑里，保存在C:/用户/*你的用户名/视频/捕获 里了

  5. Alt + Shift + S 这是win10给你的牛逼的截图

  6. Ctrl + Alt + A 这个是QQ给你的截图，在电脑打开QQ之后才能用

  7. Alt + A 这个是微信给你的截图，在电脑打开微信之后才能用

快速操作

  • win + E 打开文件资源管理器

  • win + D 快速显示桌面

  • win + M 快速最小化所有窗口（相对于上一个会有一个动画效果）

  • win + ←→ 分屏操作

  • win + ↑↓ 最大化最小化窗口

  • Alt + F4 快速关闭当前软件

  • Alt + Tab 两个程序交换

  • Ctrl + 拖文件 复制文件

装逼操作

  • Shift + 右键 打开Windows PowerShell Alt → 空格 → R 等同

  • Shift + ^ 在中文状态下输入省略号

  • win + + 打开放大镜

  • win + Esc 关闭放大镜
